# sysmon-cli
A simple command-line system monitor

## What it does
sysmon-cli is a tool that prints real-time system metrics and notifications to the console. It's a focused, single-purpose tool that helps me keep an eye on my machine's performance.

## Install
You can install sysmon-cli using Cargo:

```bash
cargo install sysmon-cli
```

## Usage
To run sysmon-cli, simply execute the binary in your terminal:

```bash
sysmon-cli
```

## Build from source
To build sysmon-cli from source, clone this repository and run the following command:

```bash
cargo build
```

## Run tests
There are no tests for this project yet.

## Project structure
- `src/main.rs`: The entry point of the program.
- `src/lib.rs`: A library module containing utility functions.
- `Cargo.toml`: The Cargo configuration file.
